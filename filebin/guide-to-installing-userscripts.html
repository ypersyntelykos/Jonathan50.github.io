Guide to installing userscripts

Firefox:
Install the Greasemonkey add on, download the userscript and install it.
Google Chrome:
Go to the extensions page, download the userscript and drag it onto the page.
Safari:
Install Greasemonkey to safari. Download the userscript and install.
IE/Internet Explorer:
Install Greasemonkey to ie. Download the userscript and install.
